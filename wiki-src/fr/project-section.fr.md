# Section Projet

La section « Projet » est le cœur de la ValueApp.
Elle relie la description initiale aux calculs de prestations.

Les principes sont expliqués plus en détail dans
[le fonctionnement de l’application](how-it-works.fr.md).

## Contenu de la section

- Quantité déterminante
- Usages
- Évaluation de la complexité

Ces éléments influencent directement les résultats présentés dans les
[exemples](examples.fr.md).
