# Comment fonctionne la ValueApp

Cette page décrit la logique interne de la ValueApp,
depuis la saisie des données jusqu’aux résultats.

Elle complète les
[Premiers pas](getting-started.html) et le
[tutoriel](tutorial.fr.md).

## Flux de données

Les données saisies sont combinées à des modèles de référence,
puis ajustées à l’aide de facteurs décrits dans les
[définitions](definitions.fr.md).

## Résultats

Les résultats doivent être interprétés dans leur contexte,
comme illustré dans les
[exemples](examples.fr.md).
