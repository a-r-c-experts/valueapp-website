# Tutoriel

Ce tutoriel explique étape par étape comment utiliser la ValueApp,
en s’appuyant sur les concepts décrits dans les
[définitions](definitions.fr.md).

## Étapes principales

1. Décrire le projet
2. Définir la quantité et la complexité
3. Sélectionner les prestations
4. Examiner le résumé

Chaque étape correspond à une partie du
[fonctionnement de l’application](how-it-works.fr.md).
